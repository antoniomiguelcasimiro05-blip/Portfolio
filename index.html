Digital Task 5 – Calculus in Motion

In this task, I learned how calculus can describe how things move and change over time. Instead of just looking at numbers, it helps explain real situations like a car speeding up or slowing down.

Velocity tells us how fast something is moving, while acceleration shows how that speed is changing. For example, if a car is getting faster, its acceleration is positive. If it slows down, the acceleration becomes negative.

If we have a simple function like s(t) = t², we can find how fast it’s moving by getting the derivative, which gives v(t) = 2t. This means the speed keeps increasing as time goes on. Taking another derivative gives a(t) = 2, so the acceleration stays constant.

This connects to real life because vehicles, machines, and even rides like Ferris wheels all depend on smooth changes in motion. Engineers use these ideas to make sure everything moves safely and properly.

Reflection

This activity helped me understand that calculus is really about change. It made the topic feel more useful because I can see how it applies to things around me, not just in math problems.
